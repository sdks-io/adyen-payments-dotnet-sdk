
# Ship Address Usage Ind Enum

Indicates when the shipping address used for this transaction was first used with the 3DS Requestor.
Allowed values:

* **01** — This transaction
* **02** — Less than 30 days
* **03** — 30–60 days
* **04** — More than 60 days
