
# Trans Type Enum

Identifies the type of transaction being authenticated. Length: 2 characters. Allowed values:

* **01** — Goods/Service Purchase
* **03** — Check Acceptance
* **10** — Account Funding
* **11** — Quasi-Cash Transaction
* **28** — Prepaid Activation and Load
